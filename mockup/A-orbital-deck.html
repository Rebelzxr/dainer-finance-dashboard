// DESIGN NOTES

The orbital
vocabulary.

TYPE
Inter Thin + IBM Plex Mono.
Display = Inter 100/200 at massive scale. Mono = IBM Plex Mono. No serif. The "editorial" instinct is replaced with NASA-mission-control restraint. Numbers are oversized + tabular.
COLOR
Void + ice-cyan only.
Pure #000000. One signature: ice-cyan #B8F0FF — Hubble-image blue. Light mode flips to #FBFCFD + deep-teal #006B7A. No second accent. No gradient anywhere.
SPACE
Negative space is the design.
120px section padding, 80px gaps. Picks are far apart, not crammed. The void between cards is doing the work. Star-field backdrop only at extreme low opacity. Corner brackets mark every pick — like coordinate ticks.
